Wasm vs JS Solver Benchmark

What is being measured

This benchmark compares two implementations of a Mindlin plate finite-element solver:

  • JavaScript — banded Cholesky (src/solver/mindlin.js).
  • WebAssembly — banded Cholesky, compiled from C++ via Emscripten (wasm/solver.cpp).

Both use the same algorithm (banded stiffness matrix + Cholesky factorisation). The comparison measures implementation performance: V8 JIT vs ahead-of-time-compiled WASM.

For each mesh size n×n, the solver assembles and solves a system with DOF = 3 (n+1)² (three unknowns per node: deflection w, and two rotations θx, θy). The reported time covers assembly + solve only — no rendering, no DOM updates. Each mesh size is run 2 warm-up iterations (discarded) followed by 5 timed iterations; the chart shows the average of the timed runs.

The test problem is a simply-supported 20 m × 20 m plate under 1 kPa uniform load.
